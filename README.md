# NordFit

Landing page for a premium fitness studio. High-energy dark design with lime green accents, Geist typography, trainer profiles, pricing plans, and trial booking form.

## Tech Stack

- **Next.js 16** — App Router, Turbopack
- **Tailwind CSS v4** — `@theme` directive for custom design tokens
- **Framer Motion** — Scroll-triggered animations
- **Lucide React** — Icon system

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
components/
  Header.tsx        — Fixed navbar with mobile hamburger menu
  Hero.tsx          — Full-screen hero with CTA
  Programs.tsx      — Training programs overview
  Trainers.tsx      — Trainer profiles
  Pricing.tsx       — Membership pricing plans
  BookingForm.tsx   — Trial session booking form
  Footer.tsx        — Site footer with navigation
```
